# **App Name**: AdWise AI

## Core Features:

- User Authentication: Secure user authentication with email, password, and phone number options. Includes a verification system and a welcome bonus of $4 for new users.
- Automated Ad Creation: AI-powered tool to automatically create ad campaigns across multiple platforms (Google, TikTok, Facebook, Snapchat) with realistic performance simulations.
- Smart Ad Review: AI-driven system that reviews and activates ads within 10-15 minutes, ensuring compliance and relevance.
- Automated Site Management: AI "tool" that autonomously develops the website by creating new pages and articles, analyzing competitor SEO, and generating top-ranking content.
- Financial Transactions: Integrated financial system that supports balance top-ups with a 20% discount, a 20% commission via referral links, and multiple payment methods (Binance, e-wallets).
- Automated Marketing: AI driven "tool" that automates lead generation, client communication, payment process management, and campaign monitoring.
- Agency Subscriptions: Subscription-based service for agencies that offers unlimited ad accounts, anti-closure protection, global targeting, and multi-platform support.

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2) to convey trust and innovation, inspired by the efficiency of automated advertising.
- Background color: Light grey (#F5F5F5) to provide a clean and modern backdrop that emphasizes content.
- Accent color: Bright cyan (#00FFFF) to draw attention to key interactive elements and AI functionalities.
- Body and headline font: 'Inter', a sans-serif font for a modern and neutral aesthetic, suitable for both headlines and body text.
- Clean, modern icons to represent different advertising features and functionalities.
- Responsive and intuitive layout with clear information architecture to enhance user experience.
- Subtle animations and transitions to provide dynamic feedback and guide users through the platform.
